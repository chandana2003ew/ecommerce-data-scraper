# 🛍️ E-Commerce Product Scraper

A Python script that scrapes product information such as **name**, **price**, and **rating** from an e-commerce website and stores the data in a CSV file.

---

## 🎯 Task 4 – SkillCraft Internship

> Create a program that extracts product information such as names, price, and ratings from an online e-commerce website and stores the data in a structured format like CSV file.

---

## 🌐 Target Website

This project uses [Books to Scrape](http://books.toscrape.com) – a safe demo site for practicing web scraping.

---

## 🧩 Features

- Scrapes:
  - Product title
  - Price
  - Rating
- Saves the data into a `products.csv` file
